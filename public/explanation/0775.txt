**正确选项解释**：
根据自由空间路径损耗公式 $ L = 32.4 + 20\log_{10}(d) + 20\log_{10}(f) $，其中 $ d $ 是距离（单位：km），$ f $ 是频率（单位：MHz）。

初始情况：
- $ d = 100 \, \text{km} $，$ f = 145 \, \text{MHz} $，发射功率为 1W（即 0dBm）。
- 路径损耗 $ L = 32.4 + 20\log_{10}(100) + 20\log_{10}(145) = 32.4 + 40 + 43.2 = 115.6 \, \text{dB} $。
- 接收功率 = 发射功率 - 路径损耗 = 0dBm - 115.6dB = -115.6dBm（与题目一致）。

当距离变为 500km 时：
- $ d = 500 \, \text{km} $，$ f = 145 \, \text{MHz} $。
- 路径损耗 $ L = 32.4 + 20\log_{10}(500) + 20\log_{10}(145) = 32.4 + 54 + 43.2 = 129.6 \, \text{dB} $。
- 接收功率 = 0dBm - 129.6dB = -129.6dBm。

因此，接收信号功率变为 **-129.6dBm**，选项A正确。

**错误选项解释**：
- 选项B（-121.6dBm）：低估了路径损耗，不符合自由空间传播模型计算结果。
- 选项C（-125.1dBm）：计算方式不正确或近似值有误。
- 选项D（-135.6dBm）：路径损耗过大，不符合理论计算结果。
